# SQL Server Data Warehouse Portfolio Project

This project demonstrates the design and development of a SQL Server Data Warehouse using ETL processes, dimensional data modelling, and data analytics.

The project involves extracting raw data from source systems, transforming and cleaning data, and loading it into a structured data warehouse environment. A Star Schema data model is implemented using Fact and Dimension tables to support efficient reporting and analysis.

The goal of this project is to showcase skills in data engineering, SQL development, data modelling, and business intelligence reporting.

## Technologies Used

- Microsoft SQL Server
- ETL Processes
- Data Modelling (Star Schema)
